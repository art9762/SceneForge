# Документация SceneForge

Этот каталог содержит сводную документацию по SceneForge. Она подготовлена на основе `CLAUDE.md` и сверена с текущим кодом репозитория.

| Документ | Что внутри |

## Текущее состояние интерфейсов

- CLI: основной и надёжный интерфейс.
- FastAPI: можно использовать для создания проекта, polling состояния, выбора концепции и чтения финальных файлов.
- Web UI: ранний прототип; фронтенд-страница проекта требует синхронизации с формой ответа API.
